概述

​ shift 命令用于对参数进行左移，可以在循环遍历控制参数数的递减，也可以去掉左边不需要的参数。shift 命令每执行一次，参数数减一，参数位置依次左移一位，比如之前的 $1 被清除，$2变成$1。同理 shift n 表示前n位参数被清除。

例子

输出：

aaa aaaf ddd xxx eee
aaaf ddd xxx eee
aaaf
ddd
xxx
eee
